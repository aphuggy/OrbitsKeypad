# Mylaps Orbits Control Keypad
Tested on Mylaps v5.10.13
No known defects



Pin 1 = F5      (Green)
Pin 2 = F9      (Yellow - NC)
Pin 3 = F8      (Checker)
Pin 4 = F10     (Manual Passing)

Connect one side of switch to ESP pin and the other to GND. ESP has built-in pullup resistors.
